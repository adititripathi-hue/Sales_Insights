# Sales Insights Dashboard

![Dashboard Screenshot] (Dashboard.png)



## Overview

This project is an interactive **Sales Insights Dashboard** built in **Power BI** to analyze sales performance, customer trends, product demand, and business growth metrics. The dashboard transforms raw sales data into meaningful visual insights that support data-driven decision-making.

## Objectives

* Monitor overall sales performance
* Identify high-performing products and categories
* Track sales trends over time
* Analyze customer and transaction patterns
* Support business decisions through visual analytics

## Tools & Technologies

* **Power BI**
* Data Modeling
* Data Visualization
* DAX (Data Analysis Expressions)
* Interactive Filters & Slicers

## Dashboard Features

* KPI cards for quick performance monitoring
* Sales trend analysis using line and area charts
* Product and category performance comparison
* Geographic sales visualization through maps
* Funnel analysis for business insights
* Interactive filtering using slicers
* Detailed tabular reporting for deeper analysis

## Key Insights Generated

* Top-performing products and sales segments
* Revenue trends across different periods
* Customer purchasing behavior patterns
* Regional sales distribution
* Sales contribution by category and product type

This dashboard provides a centralized view of sales performance, enabling stakeholders to quickly identify trends, monitor key metrics, and make informed business decisions based on data.
